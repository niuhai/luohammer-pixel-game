# 真还传 · 上线部署指南（手把手版）

---

## 🚀 阶段一：今天就能上线（10分钟，临时链接）

> 不需要花钱，不需要备案，马上拿到公网链接分享给朋友测试。

### 步骤1：构建生产版本

在项目目录打开终端，执行：

```bash
cd e:\ownWorkPlace\罗的十字路口\luohammer-pixel-game
npm run build
```

构建成功后，会生成 `dist/` 文件夹，这就是要上线的全部文件。

### 步骤2：拖拽部署到Netlify

1. 浏览器打开：https://app.netlify.com/drop
2. 把 `dist` 文件夹直接拖到页面中间的虚线框里
3. 等30秒，自动部署完成
4. 页面会显示你的临时链接，类似：`https://xxx-xxx-xxx.netlify.app`

> ⚠️ 这个链接国内访问可能偶尔慢或打不开，用来测试没问题，正式上线用阶段三。

## 💰 阶段二：准备国内正式部署（第1天操作）

### 需要准备
- 手机号
- 身份证（备案需要上传正反面）
- 支付宝/微信（买域名用）

### 步骤1：注册阿里云账号

1. 打开 https://www.aliyun.com/
2. 点击右上角「免费注册」
3. 用手机号注册，完成实名认证（必须）

### 步骤2：购买域名

1. 登录后进入：https://wanwang.aliyun.com/domain
2. 在搜索框输入你想要的域名，比如 `luohammer` 或 `zhenhuan`
3. 选一个 `.com` 后缀的，价格约55元/年
4. 加入清单，结算购买
5. 购买时勾选「开启域名隐私保护」（免费，防止个人信息泄露）

> 💡 域名建议：简短好记，和游戏相关即可，比如 luohammer.com、zhenhuanzhuan.com 之类的。

### 步骤3：提交ICP备案（关键！）

> 备案是国内网站必须的，免费，阿里云APP上操作，全程7-15天。

1. 购买域名后，进入阿里云备案页面：https://beian.aliyun.com/
2. 点击「开始备案」
3. 按提示填写信息：
   - 产品类型：选「阿里云」→「云服务器ECS/OSS」（如果提示需要服务器，选OSS备案即可）
   - 如果OSS备案需要服务器，可以先买一台最便宜的轻量应用服务器（约30元/月，备案后可以退订只留OSS）
4. 下载「阿里云」APP，手机端完成：
   - 上传身份证正反面
   - 人脸识别
   - 填写网站信息（网站名称：真还传；网站类型：个人博客/作品展示；内容：互动叙事游戏）
   - 签署核验单
5. 提交后等待审核：
   - 阿里云初审：1-2天
   - 管局审核：5-10天
   - 审核通过会发短信通知

> 📱 **备案期间你可以做的事**：继续完善游戏、做测试、修bug。

## 🎉 阶段三：备案通过后，正式上线（10分钟）

备案通过后，按以下步骤配置：

### 步骤1：开通OSS对象存储

1. 登录阿里云控制台：https://oss.console.aliyun.com/
2. 点击「创建Bucket」
3. 填写：
   - Bucket名称：`luohammer-game`（或你喜欢的名字）
   - 地域：选离你最近的（如「华东1（杭州）」）
   - 存储类型：标准存储
   - 读写权限：**公共读** ⚠️（必须选这个，否则网页打不开）
   - 其他默认，点确定

### 步骤2：上传dist文件夹到OSS

#### 方式A：控制台手动上传（简单）
1. 进入刚创建的Bucket
2. 点击「文件管理」→「上传文件」
3. 选择 `dist` 文件夹里的**所有文件**（包括index.html、assets文件夹、sw.js等）
4. 上传

#### 方式B：脚本自动上传（推荐，后续更新方便）
> 项目里已经准备好了部署脚本，等你拿到AccessKey后运行 `npm run deploy` 一键上传。

### 步骤3：配置静态网站托管

1. Bucket页面 → 点击「基础设置」→「静态页面」
2. 开启静态网站托管
3. 默认首页：`index.html`
4. 默认404页面：`index.html`（SPA必须这样配，否则刷新会404）
5. 保存

### 步骤4：开通CDN并绑定域名

1. 进入CDN控制台：https://cdn.console.aliyun.com/
2. 点击「添加域名」
3. 填写：
   - 加速域名：`你买的域名`（如 `game.yourdomain.com`）
   - 业务类型：图片小文件
   - 源站类型：OSS域名
   - 源站：下拉选择你刚创建的Bucket
   - 端口：443
4. 下一步，等待CNAME分配（1-2分钟）

### 步骤5：配置域名解析

1. 进入阿里云域名解析：https://dns.console.aliyun.com/
2. 找到你买的域名，点击「解析设置」
3. 添加记录：
   - 记录类型：CNAME
   - 主机记录：`game`（如果用game子域名）
   - 记录值：CDN分配给你的CNAME地址（在CDN控制台复制）
   - TTL：10分钟
4. 保存

### 步骤6：配置HTTPS（免费证书）

1. 进入SSL证书控制台：https://yundun.console.aliyun.com/?p=cas
2. 点击「免费证书」→「创建证书」
3. 填写你的域名，按提示验证域名所有权
4. 证书签发后，到CDN控制台→你的域名→HTTPS配置
5. 选择「已签发证书」，选择刚申请的免费证书
6. 开启「强制HTTPS跳转」

### 步骤7：配置CDN缓存规则

在CDN控制台→你的域名→缓存配置：

1. 添加缓存规则：
   - 类型：目录，路径：`/assets/`，缓存时间：1年
   - 类型：文件，路径：`sw.js`，缓存时间：0（不缓存）
   - 类型：文件，路径：`index.html`，缓存时间：0（不缓存）
   - 类型：文件，路径：`manifest.json`，缓存时间：1小时
   - 其他默认（缓存10分钟）

## 🔄 后续更新游戏

每次修改游戏后，更新上线只需3步：

```bash
# 1. 重新构建
npm run build

# 2. 上传到OSS（脚本方式）
npm run deploy

# 3. 如果CDN缓存了旧版index.html，去CDN控制台刷新缓存即可
```

## 📱 微信分享优化（可选）

备案完成、域名解析好后：
1. 确保 `og-image.png` 在根目录可访问
2. 检查 `index.html` 里的og标签图片URL是否为绝对路径：
   ```html
   <meta property="og:image" content="https://game.yourdomain.com/og-image.png">
   ```
3. 微信里打开链接，右上角「...」→「分享给朋友」应该能看到卡片

## ❓ 常见问题

**Q：备案期间我必须买服务器吗？**
A：用OSS产品备案即可，不需要单独买服务器。如果阿里云要求，可以买最便宜的轻量服务器（约30元/月），备案通过后退订。

**Q：后续做第二个项目需要再买域名吗？**
A：不用！一个域名可以加无数个子域名。比如：
- 真还传 → `game.yourdomain.com`
- 项目B → `b.yourdomain.com`
- 项目C → `c.yourdomain.com`

**Q：我不想备案，有办法国内访问吗？**
A：可以买香港节点的服务器（不需要备案），但速度比国内CDN慢，微信分享可能有问题。不推荐。

**Q：费用什么时候扣？**
A：域名一次性年付，OSS和CDN是按量后付费（每月从账户余额扣），用多少扣多少。

# 真还传·创业模拟器 — 项目背景文档

## 〇、项目灵魂

> **选择没有对错，只有不同的人生。**

这个项目的出发点不是"做一个好玩的游戏"，而是**有感而发**。

罗远是一个传奇人物。他的人生充满选择——砸冰箱、做手机、欠6亿、直播还债——每一个选择在当下看都未必是最优解，甚至很多选择从利益最大化的角度是"错"的。但正是这些"错"的选择，定义了他是谁。

- 砸冰箱——商业上愚蠢，但定义了"较真"
- 做锤子手机——烧光所有钱，但"天生骄傲"不是空话
- 拒绝烂广告——少赚几千万，但保住了信任
- 直播还债——放下身段，但"真还传"反而赢得更多尊重

**如果用利益最大化来衡量，老罗的很多选择都是"错"的。但正是这些"错"的选择，让他成为了老罗。**

所以这个游戏的核心不是"选对得分、选错扣分"，而是让玩家**感受每个选择的重量**——你选的不是对错，你选的是你想成为什么样的人。

### 设计哲学

1. **不评判选择** — 没有标准答案，每条路都有风景
2. **选择塑造身份** — 属性不是分数，而是"你是什么样的人"的映射
3. **代价即意义** — 放弃的东西定义了你坚持的东西
4. **所有结局都值得尊重** — "传奇"不比"幸存者"更好，只是不同
5. **重玩不是为了赢，而是为了体验另一种人生**

## 一、项目概述

**项目名称**：真还传·创业模拟器（Luohammer: Crossroads of Destiny）

**项目定位**：以罗远人生经历为背景的2D像素风互动养成游戏。玩家在关键人生节点做出选择，体验从延边少年到6亿负债再到真还传的跌宕人生。

**产品形态**：H5网页游戏，无需下载，微信生态传播

**核心卖点**：
- 罗远IP自带流量和话题性
- 像素风极简美学，中国式家长的养成感
- 人生重开模拟器的随机性+隐形守护者的叙事深度
- 每局不同，重玩价值高

## 二、对标产品分析

### 2.1 人生重开模拟器

**成功要素**：
- 天赋抽取系统：开局3选1天赋，决定初始属性和特殊能力
- 随机事件池：每局事件不同，高重玩性
- 属性驱动：颜值/智力/体质/家境影响事件触发和结局
- 极简呈现：纯文字+emoji，零美术成本，但极具代入感
- 社交传播：截图分享"我的人生"，病毒式传播

**可借鉴**：
- 天赋抽取开局
- 随机事件池机制
- 属性阈值触发隐藏事件
- 结局截图分享

**需避免**：
- 纯文字太单调，需要像素风视觉加持
- 随机性过强导致玩家失去掌控感

### 2.2 中国式家长

**成功要素**：
- 养成循环：每回合分配行动点→属性增减→阶段考试/事件
- 压力系统：属性不是只增不减，压力过高会崩溃
- 代际传承：一代结局影响二代初始条件
- 情感共鸣：中国式家庭教育的集体记忆
- 卡通手绘风格：温暖但不幼稚

**可借鉴**：
- 属性不只是加减，有压力/崩溃机制
- 阶段性结算（对应老罗的人生阶段）
- 每个选择都有代价，不是"选哪个都行"
- 情感共鸣：创业者的集体焦虑

**需避免**：
- 代际传承不适合本IP（老罗只有一个人）
- 过于复杂的行动点系统

### 2.3 隐形守护者 / 恋爱模拟器

**成功要素**：
- 选择有远期后果：第1章的选择影响第8章的结局
- 隐藏属性：好感度/立场值不直接显示，通过剧情暗示
- 多周目探索：不同选择解锁完全不同的故事线
- 情感投入：角色有血有肉，选择有道德重量

**可借鉴**：
- 选择的远期后果（如：早期选"妥协"→后期失去盟友）
- 隐藏属性（如：公众信任度，不直接显示但影响结局）
- 关键选择的道德困境（如：为了还债是否接烂广告）
- 多周目探索动力

**需避免**：
- 真人影视成本太高
- 线性叙事缺乏随机性

## 三、核心玩法设计

### 3.1 游戏循环

```
天赋抽取 → 属性初始化 → 阶段推进
    ↓
遇到事件 → 做出选择 → 属性变化 + 远期标记
    ↓
随机事件插入 → 属性阈值检测 → 触发/跳过隐藏事件
    ↓
阶段结算 → 下一阶段 → ... → 结局
```

### 3.2 属性系统

| 属性 | 范围 | 说明 | 影响 |
|------|------|------|------|
| 理想主义 | 0-10 | 老罗的执念和坚持 | 高→解锁"传奇"路线；低→解锁"务实"路线 |
| 财富 | 0-10 | 当前经济状况 | 高→解锁投资选项；低→被迫接受妥协 |
| 名声 | 0-10 | 公众形象 | 高→获得支持；低→被舆论反噬 |
| 翻车次数 | 0+ | 失败经历累计 | 每次翻车+1，高翻车解锁"浴火重生" |
| 压力 | 0-10 | 隐藏属性 | 过高→强制做出非理性选择；归零→"看破红尘" |
| 公众信任 | 0-10 | 隐藏属性 | 影响"真还传"结局是否触发 |

### 3.3 天赋系统（开局抽取）

每次开局随机3选1：

| 天赋 | 效果 | 描述 |
|------|------|------|
| 嘴炮王者 | 理想主义+2，名声+1 | "我这个人就是嘴欠" |
| 偏执狂 | 理想主义+3，压力+2 | "工匠精神就是死磕" |
| 社交达人 | 名声+2，财富+1 | "朋友多了路好走" |
| 厚脸皮 | 压力上限+3 | "被骂习惯了" |
| 商业嗅觉 | 财富+2，理想主义-1 | "闻到钱的味道" |
| 锦鲤体质 | 随机事件偏向正面 | "运气也是实力" |
| 孤注一掷 | 翻车惩罚减半，成功奖励翻倍 | "all in" |
| 老罗附体 | 全属性+1 | "天生骄傲"（稀有） |

### 3.4 随机事件池

每个阶段有固定剧情节点 + 随机事件插入：

**创业期随机事件**：
- 投资人突然撤资（财富-2）
- 员工集体离职（压力+2）
- 产品意外爆火（名声+2，财富+1）
- 被大厂抄袭（理想主义-1，压力+1）

**还债期随机事件**：
- 直播间被封（名声-2）
- 债主上门催债（压力+3）
- 粉丝自发帮忙宣传（名声+1）
- 意外获得投资机会（财富+2）

### 3.5 选择远期后果

选择不只是影响下个节点，而是标记远期影响：

| 阶段 | 选择 | 远期后果 |
|------|------|---------|
| 新东方时期 | "公开批评体制" | → 后期获得公众同情+1 |
| 锤子时期 | "坚持做高端" | → 财富-2但理想主义+2，后期解锁"工匠传奇"结局 |
| 锤子时期 | "妥协做中端" | → 财富+1但理想主义-1，后期失去核心粉丝 |
| 直播还债 | "接所有广告" | → 财富+3但名声-2，公众信任-2 |
| 直播还债 | "只接靠谱的" | → 财富+1但名声+1，公众信任+1 |
| 退网宣言 | "真的退网" | → 锁死部分路线，但解锁"隐士"结局 |
| 退网宣言 | "过几天就回来" | → 名声-1，但保留所有路线 |

### 3.6 阶段系统

| 阶段 | 章节 | 核心冲突 |
|------|------|---------|
| 少年 | 1-3 | 退学、自学、去新东方 |
| 名师 | 4-6 | 新东方讲课、砸冰箱、维权 |
| 创业 | 7-12 | 锤子科技、做手机、T1/T2 |
| 至暗 | 13-16 | 锤子倒闭、欠债6亿、被限高 |
| 还债 | 17-20 | 直播带货、真还传、退网 |
| 新生 | 21-23 | 播客、AI转型、新可能 |

### 3.7 结局体系（12+结局）

**设计原则：所有结局都值得尊重，没有"好结局"和"坏结局"，只有"不同的人生"。**

| 结局 | 触发条件 | 主题 | 尊重点 |
|------|---------|------|--------|
| 传奇 | 理想主义≥8 + 翻车≥2 + 财富<3 | 偏执到底，终成传奇 | 坚持的代价是孤独，但孤独是传奇的门票 |
| 大亨 | 财富≥8 + 理想主义<3 | 放弃理想，拥抱资本 | 现实不是妥协，是另一种清醒 |
| 战士 | 理想主义≥7 + 名声≥7 + 翻车≥2 | 屡败屡战，公众英雄 | 跌倒不可怕，站起来才了不起 |
| 替罪羊 | 名声<2 + 翻车≥3 | 众叛亲离 | 有时候不是你做错了，是时代没站在你这边 |
| 平衡 | 全属性6+ + 翻车=0 | 完美人生 | 最难达成的结局——因为人生没有完美 |
| 理性 | 理想主义≥5 + 财富≤3 + 翻车≥3 | 看透本质，选择务实 | 理性不是冷血，是另一种勇气 |
| 供应链 | 财富≥4 + 理想主义≤3 + 翻车≥2 | 被现实磨平 | 大多数人的结局，不代表失败 |
| 脱口秀之星 | 名声≥8 + 理想主义≥5 | 回归表达者本色 | 天生就是说话的人 |
| AI远见者 | 财富≥7 + 名声≥5 + 理想主义≤4 | 赶上新时代 | 活着就要向前看 |
| 浴火重生 | 理想主义≥7 + 翻车≥3 + 财富≥4 | 真还传本传 | 最老罗的结局——跌到底，再站起来 |
| 隐士 | 压力归零 + 退网选择 | 看破红尘 | 放下也是一种选择 |
| 和平 | 全属性3-6 + 翻车≤2 | 平凡但安稳 | 平凡不是平庸 |
| 幸存者 | 翻车≥4 | 活着就好 | 活着本身就是胜利 |
| 工匠传奇 | 理想主义≥9 + "坚持做高端"标记 | 像素级完美主义 | 极致的代价是极致的孤独 |

## 四、视觉风格

### 4.1 核心风格

**像素风极简** — 不是复古8-bit，而是用像素语言表达现代感。

参考：
- **中国式家长**的温暖卡通感 → 我们用像素风实现同样的温度
- **Downwell**的极简配色 → 我们用暗色底+金色强调
- **Celeste**的像素情感表达 → 角色虽小但表情丰富

### 4.2 色板

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景 | #0A0A0A | 近纯黑，沉浸感 |
| 面板 | #12122A | 深蓝黑，层次感 |
| 强调 | #F0C040 | 金色，老罗的"天生骄傲" |
| 文字 | #E8D5A3 | 暖白，可读性 |
| 暗文字 | #8A7A5A | 暗金，辅助信息 |
| 危险 | #E04040 | 红色，翻车/警告 |
| 成功 | #40C040 | 绿色，正面反馈 |

### 4.3 角色设计

- 像素风老罗，核心辨识特征：黑框眼镜、寸头、黑T恤
- 8种姿态：站立/坐着/说话/愤怒/沮丧/直播/年轻/中年
- 尺寸：画面主体，占50%+高度，视觉小说立绘风格

### 4.4 场景设计

- 12种场景类型，每种有独特氛围
- 暗色调为主，金色光效点缀
- 不追求写实，追求氛围和辨识度

## 五、技术架构

### 5.1 技术栈

| 层 | 技术 | 说明 |
|----|------|------|
| 引擎 | Phaser 3.80 | Canvas渲染，像素风友好 |
| 构建 | Vite 5 | 快速开发+构建 |
| 渲染 | Canvas 2D | pixelArt模式，无WebGL依赖 |
| 画布 | 800×500 | Phaser.Scale.FIT自适应 |
| 部署 | Cloudflare Pages | 静态托管，全球CDN |

### 5.2 文件结构

```
src/
├── main.js              # Phaser入口
├── config.js            # 全局配置（色板/字体/场景枚举）
├── scenes/
│   ├── BootScene.js     # 标题界面
│   ├── GameScene.js     # 游戏主逻辑
│   └── EndingScene.js   # 结局界面
├── systems/
│   ├── PixelRenderer.js # 渲染引擎（角色/场景/特效）
│   ├── DialogSystem.js  # 对话框系统
│   ├── ChoiceSystem.js  # 选项系统
│   ├── StatsSystem.js   # 属性面板
│   ├── Transition.js    # 过渡动画
│   ├── AudioSystem.js   # 音效系统
│   └── SaveSystem.js    # 存档系统
├── ui/
│   ├── AchievementPopup.js
│   └── HistoryCard.js
└── data/
    └── story.js         # 剧情数据（70节点+12结局）
```

### 5.3 当前depth层级规范

| depth | 内容 |
|-------|------|
| 0 | 背景精灵/背景graphics |
| 50 | 角色精灵 |
| 100 | 前景特效(粒子/闪光) |
| 200 | 对话框容器 |
| 300 | 选项按钮容器 |
| 400 | HUD(属性面板/章节/进度条/音量) |
| 1000 | 过渡遮罩 |

## 六、目标用户

### 6.1 核心用户

- **年龄**：20-35岁
- **特征**：互联网原住民，熟悉老罗IP，玩过人生重开模拟器/中国式家长
- **场景**：通勤/摸鱼/睡前，碎片时间
- **动机**：好奇"如果我是老罗会怎么选"+ 社交分享"我的人生结局"

### 6.2 传播路径

```
微信朋友圈/群聊分享 → 扫码/点链接即玩 → 3分钟上手 → 10分钟一局
→ 截图结局分享 → 病毒传播
```

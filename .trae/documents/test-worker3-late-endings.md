# 工作者 3：后半程剧情 + 结局 + 存档测试

## 项目背景
这是一个以罗永浩人生经历为背景的 2D 像素风互动视觉小说游戏。
游戏运行在 http://localhost:62619（如果端口变了，先用 `cd e:\ownWorkPlace\罗的十字路口\luohammer-pixel-game && npx serve dist` 启动）

## 你的任务
用浏览器真实打开游戏，重点测试第 15-20 章、结局界面、存档系统。

## 测试环境准备
1. 打开 Chrome 浏览器
2. 按 F12 打开 DevTools
3. 打开 http://localhost:62619
4. 快速推进到第 15 章（锤子倒闭）

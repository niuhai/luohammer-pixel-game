# 豆包生图提示词 v2（角色优化版）

## 问题分析

豆包不认识"罗永浩"，所以角色生成效果差。需要把外貌拆成**AI 能理解的视觉元素**。

## 老罗外貌拆解（给 AI 的精确描述）

| 部位 | 精确描述 | 重要性 |
|---|---|---|
| **眼镜** | 粗黑框、方形、镜框很厚（像早期 Geek 风格）、镜片反光 | ⭐⭐⭐ 最关键 |
| **脸型** | 圆脸、下巴圆、脸颊有肉、不是尖脸 | ⭐⭐⭐ |
| **发型** | 极短寸头、几乎贴头皮、发际线较高、两侧露出头皮 | ⭐⭐⭐ |
| **体型** | 微胖、肩膀宽、肚子微凸、不是瘦子 | ⭐⭐ |
| **穿着** | 纯黑圆领T恤、无图案、无logo | ⭐⭐ |
| **表情** | 坚定/愤怒时眉毛下压、嘴微张 | ⭐ |

## 关键技巧

1. **不要提"罗永浩"三个字** — AI 不认识
2. **眼镜描述要详细** — "粗黑框方形厚眼镜，镜框黑色很粗，镜片有反光"
3. **强调圆脸微胖** — 不是瘦子，不是尖脸
4. **强调寸头** — "极短寸头发型几乎贴头皮"
5. **强调黑T恤** — "纯黑圆领T恤无图案"
6. **加"全身像"** — 确保能看到完整体型
7. **深色纯色背景** — 避免背景干扰角色识别

## 如果还是不像

可以先生成一张**参考图**（用老罗的真实照片做图生图），然后让豆包参考那张图的风格生成像素版。

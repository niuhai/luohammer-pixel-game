# UI 工作者 3：标题界面 + 结局界面 + 成就弹窗优化

## 项目背景
这是一个以罗永浩人生经历为背景的 2D 像素风互动视觉小说游戏。
技术栈：Phaser 3.80 + Vite 5，纯 Canvas 渲染。
画布尺寸：800×500。
项目根目录：`e:\ownWorkPlace\罗的十字路口\luohammer-pixel-game`

## 当前问题

1. **标题界面太简单**：只有纯文字，没有动画、没有角色头像、没有氛围。
2. **结局界面太朴素**：标题直接出现，属性条无动画，无分享功能。
3. **成就弹窗太小**：200×50px，不够醒目，无闪光效果。
4. **历史卡片太简陋**：无装饰、无动画、📜 emoji 不够像素风。

## 你要修改的文件

### 1. `src/scenes/BootScene.js` — 标题界面重做

**当前问题**：标题是纯文字，无动画，无角色，无氛围。

**目标**：有动画、有角色头像、有背景粒子、有像素风按钮。

**具体要求**：

1. **标题动画**：
   - "真""还""传"三个字分别创建为独立 Text 对象
   - 初始 alpha=0，逐个淡入（间隔 300ms）
   - 淡入后加 Y 轴弹跳（Back.Out 缓动）
   - 全部显示后加 alpha 闪烁（0.85→1→0.85，2000ms 周期）

2. **副标题打字机**：
   - "创业模拟器" 逐字出现，每字 100ms
   - 延迟 1200ms（等标题先出完）

3. **像素风老罗头像**（参考 `luo-standing.png`）：
   - 位置：x=200, y=160
   - 尺寸：64×64px
   - 用 graphics.fillRect 绘制（不是加载图片）
   - 特征：圆脸、黑框眼镜、寸头、黑衬衫
   - 呼吸动画：scale 0.98→1.02 循环

4. **背景粒子**：
   - 25 个 2×2/3×3 金色半透明方块（#F0C040, alpha 0.1-0.3）
   - 从上到下缓慢飘落，速度随机（8-18 秒）
   - 到底部后回到顶部循环

5. **按钮像素风化**：
   - 四角 L 形金色装饰（和对话框统一）
   - pointerdown 时 y+2 下沉，pointerup 恢复
   - 字体：`Press Start 2P` 10px

6. **底部信息**：
   - 版本号：`Pixel Edition v2.0`
   - 版权：`© 2025 Luohammer`（6px, #2A2A3A）

### 2. `src/scenes/EndingScene.js` — 结局界面重做

**当前问题**：标题直接出现，属性条无动画，无分享功能，无氛围。

**目标**：有专属粒子、有动画、有分享、有成就感。

**具体要求**：

1. **结局专属背景粒子**（根据结局类型）：
   - legend/warrior/comeback：金色粒子从中心向外扩散
   - tycoon/ai_visionary：绿色数据流从上到下
   - scapegoat/supply_chain：红色碎片从上飘落
   - balance/peace/default：白色雪花缓慢飘落
   - 35 个粒子，tweens 循环

2. **标题动画**：
   - 从 y=-50 滑入到 y=80（Back.Out 缓动）
   - 下方画金色半透明矩形光晕（alpha 0.1）

3. **属性条动画**：
   - 初始宽度 0，tween 在 1 秒内填充到目标值
   - 每条延迟 200ms
   - 右侧显示 "x/10" 数值（`Press Start 2P` 8px）

4. **成就列表**：
   - 每个成就一行，金色文字 + emoji 图标
   - 逐个淡入（间隔 200ms）

5. **分享按钮**：
   - "再来一次"旁加"分享结局"按钮
   - 点击生成文本并复制到剪贴板：
     ```
     【真还传·创业模拟器】
     我的结局：xxx
     理想主义：x  财富：x  名声：x  翻车：x
     成就：xxx
     来试试你的人生选择！
     ```
   - 复制后显示"已复制"提示（2 秒后消失）

6. **字体**：
   - 结局标题：`Press Start 2P` 18px 金色
   - 描述/引用：`Noto Sans SC`
   - 按钮：`Press Start 2P` 10px

### 3. `src/ui/AchievementPopup.js` — 成就弹窗优化

**当前问题**：200×50px 太小，无装饰，无闪光。

**目标**：更大、更醒目、有像素风装饰。

**具体要求**：
1. **弹窗加大**：260×60px
2. **像素风边角**：四角 L 形金色装饰（和对话框统一）
3. **闪光效果**：弹窗滑入后，4px 宽白色半透明竖条从左扫到右（300ms）
4. **图标加大**：emoji 从 20px → 28px
5. **停留时间**：2500ms → 3500ms
6. **字体**：
   - "成就解锁"：`Press Start 2P` 8px
   - 成就名：`Noto Sans SC` 13px

### 4. `src/ui/HistoryCard.js` — 历史卡片优化

**当前问题**：无装饰、📜 emoji 不够像素风、无动画。

**目标**：像素风装饰 + 卷轴图标 + 动画。

**具体要求**：
1. **像素风边角**：四角 L 形金色装饰
2. **像素风卷轴图标**：用 fillRect 画 8×12 卷轴（金色边框 + 米色内容 + 横线）
3. **出现动画**：初始 alpha=0, y+20，tween 到 alpha=1, y 原位（200ms）
4. **字体**：历史文本 `Noto Sans SC` 12px，提示 `Press Start 2P` 8px

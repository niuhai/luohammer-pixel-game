# 工作者 4：移动端 + PWA + 性能 + 兼容性测试

## 项目背景
这是一个以罗永浩人生经历为背景的 2D 像素风互动视觉小说游戏。
游戏运行在 http://localhost:62619（如果端口变了，先用 `cd e:\ownWorkPlace\罗的十字路口\luohammer-pixel-game && npx serve dist` 启动）
目标部署为 H5，主要用户通过手机微信/浏览器访问。

## 你的任务
用浏览器模拟不同设备，测试移动端体验、PWA 功能、性能和兼容性。

## 测试环境准备
1. 打开 Chrome 浏览器
2. 按 F12 打开 DevTools
3. 需要在多种模拟设备下测试

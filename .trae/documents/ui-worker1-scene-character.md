# UI 工作者 1：场景绘制 + 角色绘制优化

## 项目背景
这是一个以罗永浩人生经历为背景的 2D 像素风互动视觉小说游戏。
技术栈：Phaser 3.80 + Vite 5，纯 Canvas 渲染（graphics.fillRect）。
画布尺寸：800×500。
项目根目录：`e:\ownWorkPlace\罗的十字路口\luohammer-pixel-game`

## 当前问题

1. **场景太简陋**：当前场景用简单的 fillRect 画墙面和地面，没有层次感，看起来像色块拼接而不是游戏场景。
2. **角色太小**：当前角色约 12px 宽，在 800×500 画布上几乎看不清。
3. **角色没有辨识度**：当前是通用小人，没有黑框眼镜等老罗特征。
4. **新场景缺失**：砸冰箱、脱口秀、法院三个新场景只有基础框架，缺少细节。

## 你要修改的文件

### 1. `src/systems/PixelRenderer.js` — 重写角色和场景

#### 角色部分

**当前问题**：`_drawChar` 方法中的角色太小（约 12px），没有黑框眼镜特征。

**目标**：参考 `assets/characters/luo-standing.png` 等图片，用 fillRect 绘制有辨识度的像素老罗。

**具体要求**：
1. 角色放大到 **60-70px 高**（占画布高度的 12-14%）
2. **黑框眼镜必须清晰可见**（参考图中最明显的特征）
   - 镜框：2px 粗深色矩形
   - 镜片：深色 + 白色 2×2 反光点
   - 鼻梁：2px 连接线
3. **圆脸微胖**（参考图中脸颊有肉、下巴圆）
4. **寸头**（参考图中头顶短、两侧露出头皮）
5. **黑衬衫**（参考图中有领子+纽扣，不是纯黑T恤）
6. 所有尺寸是 **4 的倍数**（GRID=4）
7. 角色外圈加 **2px 深色轮廓线**

**8 种姿态全部重绘**（参考对应图片）：
- `standing` — 参考 `luo-standing.png`
- `speaking` — 参考 `luo-speaking.png`，右手高举
- `angry` — 参考 `luo-angry.png`，双手举锤
- `depressed` — 参考 `luo-depressed.png`，低头弯腰
- `sitting` — 参考 `luo-sitting.png`
- `livestream` — 参考 `luo-livestream.png`，面对镜头
- `young` — 参考 `luo-young.png`，蓝衬衫、更瘦
- `middle` — 参考 `luo-middle.png`

#### 场景部分

**当前问题**：场景只有墙面+地面，没有前景/中景/远景分层。

**目标**：每个场景有 3 个层次（背景→中景→前景），增加细节和氛围。

**具体要求**（参考场景图和视觉规范中的色板）：

1. **classroom**：
   - 背景：深蓝色墙壁 #2A2A3E
   - 中景：黑板（#1A2A1A）+ 讲台
   - 前景：第一排课桌（大尺寸，不遮挡角色头部）
   - 细节：窗户透进微光、木纹地板纹理

2. **lecture**：
   - 背景：深色场馆 #1A1A2E
   - 中景：讲台 + 投影屏幕
   - 前景：前排观众后脑勺剪影
   - 细节：聚光灯效果

3. **office**：
   - 背景：深蓝墙壁 #2A2A3E
   - 中景：白板 + 书架
   - 前景：近景桌面边缘 + 咖啡杯
   - 细节：显示器白光、深夜氛围

4. **stage**：
   - 背景：深色 #0A0A1A
   - 中景：巨大 LED 屏幕 + 聚光灯
   - 前景：前排观众 + 手机闪光灯（白色小方块闪烁）
   - 细节：戏剧性灯光

5. **livestream**：
   - 背景：深色 #0A0A1A
   - 中景：环形补光灯 + 桌面
   - 前景：更大更近的产品展示
   - 细节：暖色灯光

6. **street**：
   - 背景：夜空 #0A0A1A + 星星
   - 中景：居民楼窗户（零星灯光）
   - 前景：路灯柱（左侧大尺寸）
   - 细节：霓虹灯招牌（粉色/红色）

7. **fridge_smash**（砸冰箱，全新）：
   - 背景：玻璃幕墙写字楼 #3A4A5E
   - 中景：3 台白色冰箱 + 碎塑料片
   - 前景：更大更近的碎塑料片
   - 细节：横幅"温和要求西门子承认并解决冰箱门问题"

8. **talkshow**（脱口秀，全新）：
   - 背景：深色 #0A0A1A
   - 中景：圆形舞台 + 红色幕布
   - 前景：舞台边缘灯光条
   - 细节："TALK SHOW" 霓虹灯牌、立式麦克风

9. **court**（法院，全新）：
   - 背景：冷色墙壁 #3A3A4E
   - 中景：木制审判桌 + 国徽
   - 前景：栏杆
   - 细节：冷色调、压抑氛围

# W1-c：新增场景绘制

## 项目背景
这是一个以罗永浩人生经历为背景的 2D 像素风互动视觉小说游戏。
技术栈：Phaser 3.80 + Vite 5，纯 Canvas 渲染（graphics.fillRect）。
画布尺寸：800×500。
项目根目录：`e:\ownWorkPlace\罗的十字路口\luohammer-pixel-game`

## 当前问题
3 个新场景（砸冰箱/脱口秀/法院）只有基础框架，缺少细节和氛围。需要从零实现这 3 个场景方法。

## 参考图和规范
- 之前豆包生成的砸冰箱/脱口秀/法院场景图
- `.trae/documents/visual-design-spec.md` — 色板

## 你要修改的文件
`src/systems/PixelRenderer.js` — 新增 3 个方法 + 在 drawScene switch 中添加 case

## 具体要求

### 1. drawFridgeSmash(g) — 砸冰箱现场

这是整个游戏最具标志性的场景之一！

```
场景描述：2011年11月20日，西门子北京总部大厦前，
罗永浩砸碎3台西门子冰箱。
色调：冷峻 + 戏剧性冲突感
```

**层次设计**：

| 层次 | 内容 | 颜色 | 说明 |
|---|---|---|---|
| 背景 | 西门子玻璃幕墙大厦 | #3A4A5E | 大面积填充，有玻璃反光条纹 |
| 天空 | 夜空 | #0A0A1A | 顶部一小块 |
| 中景 | 3台白色冰箱 | #E8E8E8 | 并排摆放，中间那台最大 |
| 中景 | 冰箱门细节 | #B0B0B0 | 门缝、把手 |
| 中景 | 横幅 | #F0F0F0 | "温和要求..."文字区 |
| 前景 | 碎塑料片 | #D0D0D0 | 更大更近的碎片 |
| 前景 | 铁锤（可选） | #808080 | 如果空间允许 |

**关键细节**：
- 大厦要有玻璃幕墙质感（水平条纹明暗交替）
- 冰箱要有门把手（小矩形）
- 横幅用红色文字区表示（红色背景 + 白色文字区）
- 碎塑料片用不规则多边形模拟（多个小 fillRect 组合）
- 整体氛围：冷峻、戏剧性、冲突感

### 2. drawTalkshow(g) — 脱口秀舞台

```
场景描述：脱口秀大会舞台，老罗首次登台表演脱口秀，
"真还传"谐音梗诞生的地方。
色调：温暖 + 聚光灯 + 轻松氛围
```

**层次设计**：

| 层次 | 内容 | 颜色 | 说明 |
|---|---|---|---|
| 背景 | 深色场馆 | #0A0A1A | 几乎全黑 |
| 中景 | 红色幕布 | #8A1A1A | 两侧垂直长条 |
| 中景 | 圆形舞台 | #2A1A1A | 中央椭圆形/圆形 |
| 中景 | 立式麦克风 | #3A3A3A | 舞台中央偏前 |
| 中景 | "TALK SHOW" 灯牌 | #FF3060 | 舞台上方，霓虹风格 |
| 前景 | 舞台边缘灯光 | #F0C040 | 底部水平光带 |
| 前景 | 观众席前排 | #0A0A1A | 底部剪影 |

**关键细节**：
- 幕布要有褶皱感（多条竖线）
- "TALK SHOW" 用红色粗字体区域表示
- 麦克风要有支架（细线 + 圆形话筒头）
- 聚光灯从上方照下来（锥形渐变）
- 整体氛围：温暖、轻松、舞台感

### 3. drawCourt(g) — 法院

```
场景描述：被限制消费令后的听证/庭审场景，
老罗面对债务问题的法律场合。
色调：冷峻 + 压抑 + 严肃
```

**层次设计**：

| 层次 | 内容 | 颜色 | 说明 |
|---|---|---|---|
| 背景 | 法庭墙壁 | #3A3A4E | 冷灰色 |
| 背景 | 国徽 | #C0A040 | 墙上中央，简化为圆+五角星 |
| 中景 | 审判桌 | #5A4A3A | 中央大桌子 |
| 中景 | 桌上文件 | #E8E8D0 | 小矩形 |
| 前景 | 栏杆 | #4A3A2A | 多根竖杆 + 横杆 |
| 地面 | 木地板 | #4A3A2A | 深色木纹 |

**关键细节**：
- 国徽简化但可识别（圆形外框 + 内部五角星）
- 审判桌要显得庄严（对称、厚重）
- 栏杆要有立体感（多根竖杆 + 2 道横杆）
- 整体色调偏冷（蓝灰 > 暖色）
- 光线从上方投射，产生阴影区域

### 代码结构

在 PixelRenderer.js 中新增 3 个方法：

```javascript
drawFridgeSmash(g) { /* ... */ }
drawTalkshow(g)   { /* ... */ }
drawCourt(g)       { /* ... */ }
```

并在 `drawScene(g, sceneType)` 的 switch 中添加：

```javascript
case 'fridge_smash':
  this.drawFridgeSmash(g);
  break;
case 'talkshow':
  this.drawTalkshow(g);
  break;
case 'court':
  this.drawCourt(g);
  break;
```

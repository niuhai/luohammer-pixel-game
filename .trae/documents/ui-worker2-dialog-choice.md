# UI 工作者 2：对话框 + 选项按钮 + 属性面板优化

## 项目背景
这是一个以罗永浩人生经历为背景的 2D 像素风互动视觉小说游戏。
技术栈：Phaser 3.80 + Vite 5，纯 Canvas 渲染。
画布尺寸：800×500。
项目根目录：`e:\ownWorkPlace\罗的十字路口\luohammer-pixel-game`

## 当前问题

1. **对话框太简陋**：纯黑底 + 2px 金色边框，像调试界面而不是游戏 UI。
2. **角色名没有层次**：名字和正文用同样排版，玩家不容易识别"谁在说话"。
3. **"点击继续"没有动画**：静态文字，玩家可能注意不到需要点击。
4. **选项按钮太朴素**：没有像素风质感，hover 效果不明显。
5. **属性条太小**：在手机上几乎看不清。
6. **字体不统一**：全部用系统 monospace，和标题界面的 Press Start 2P 不一致。

## 你要修改的文件

### 1. `src/systems/DialogSystem.js`

**当前问题**：对话框是黑底矩形，四角空，名字无下划线，提示无动画。

**目标**：改成像素风游戏对话面板。

**具体要求**：
1. **四角装饰**：加 8×8 金色 L 形装饰角（参考视觉规范）
2. **渐变背景**：从 #0A0A1A（顶）到 #12122A（底），4 条色带
3. **角色名金色下划线**：名字下方画 2px 高金色线，宽度自适应
4. **"点击继续"闪烁**：alpha 0.3→1→0.3 循环（用 scene.tweens）
5. **打字音效**：每 3 个字符播放微弱"嘟"声（scene.audio._playTone(800, 0.02, 'square', 0.03)）
6. **字体**：
   - 角色名：`Press Start 2P` 10px，金色
   - 正文：`Noto Sans SC` 13px，白色 #E8E8E8
   - 提示：`Press Start 2P` 8px，暗色 #8A7A5A

### 2. `src/systems/ChoiceSystem.js`

**当前问题**：选项按钮是纯矩形，无装饰，hover 效果弱。

**目标**：像素风按钮，有装饰角和扫描线效果。

**具体要求**：
1. **按钮边角装饰**：四角加 4×4 金色 L 形装饰
2. **选中扫描线**：hover 时画 3-4 条半透明白色水平线（间隔 4px），模拟 CRT
3. **选项滑入动画**：从下方 +40px 滑入，Back.easeOut，每个延迟 100ms
4. **键盘快捷键**：A/B/C 键选择第 1/2/3 个选项
5. **触摸反馈**：pointerdown 时 alpha=0.7，pointerup 恢复
6. **触摸区域**：按钮高度 ≥ 44px（手机标准）
7. **字体**：
   - 字母：`Press Start 2P` 12px 金色
   - 文本：`Noto Sans SC` 14px 白色

### 3. `src/systems/StatsSystem.js`

**当前问题**：属性条太小（80×6px），无图标，无数值显示。

**目标**：加大 + 加图标 + 加数值 + 动画。

**具体要求**：
1. **尺寸加大**：条宽 100px，高 10px，间距 16px
2. **像素风图标**（8×8，用 fillRect 绘制）：
   - 理想主义：火焰（红橙 #E04020）
   - 财富：金币（金色圆形 #F0C040）
   - 名声：星星（金色五角星简化）
   - 翻车：骷髅（白色简化）
3. **数值文字**：条右侧显示当前值，`Press Start 2P` 8px 金色
4. **边框**：条外加 1px 深色边框 #1A1A2E
5. **填充动画**：用 tween 从旧值平滑过渡到新值（300ms）
6. **字体**：label 用 `Noto Sans SC` 10px

## 不能做
1. 不要修改 GameScene.js（调用方）
2. 不要修改 PixelRenderer.js（另一个工作者在改）
3. 不要修改 config.js
4. 不要引入新的 import（用 scene.audio 延迟获取）
5. 不要改变方法签名（show/hide/update 等）
